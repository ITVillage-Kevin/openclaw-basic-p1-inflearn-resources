# Docs 안내

이 폴더에는 **OpenClaw.ai 기초 1편** 강의에서 제공하는 무료 가이드 문서들이 모여 있습니다.  
영상 수업에서 다 다루지 못한 설치 절차, 체크리스트, 명령어 치트시트, 프롬프트 템플릿 등을 여기서 확인할 수 있습니다.

---

## 파일명 규칙

가이드 문서는 대략 아래와 같은 이름 규칙을 따릅니다.

- `01-setup-checklist.pdf`  
  - OpenClaw.ai 설치 전 사양 및 환경 체크리스트
- `02-docker-sandbox-guide.pdf`  
  - Docker 설치 및 보안 샌드박스 구축 가이드
- `03-docker-cheatsheet.pdf`  
  - 자주 사용하는 Docker/리눅스 명령어 & 트러블슈팅 치트시트
- `04-system-prompt-template.pdf`  
  - OpenClaw 지능을 깨우는 초기 설정(System Prompt) 템플릿
- `05-file-classification-prompts.pdf`  
  - 업무 효율을 높이는 파일 자동 분류 프롬프트 모음

※ 실제 파일명은 강의에서 안내하는 규칙을 따르며, 앞의 번호(`01`, `02`, …)는 **강의 흐름 순서**를 의미합니다.

---

## 언제 어떤 문서를 보면 좋을까요?

- **설치 전에**  
  - `01-setup-checklist.pdf`  
    → 내 PC가 최소 사양을 만족하는지, 어떤 준비가 필요한지 먼저 점검할 때 사용합니다.

- **Docker와 샌드박스 환경을 만들 때**  
  - `02-docker-sandbox-guide.pdf`  
    → Docker Desktop 설치, 기본 설정, 샌드박스 개념을 다시 확인하고 싶을 때 참고합니다.

- **터미널/명령어가 헷갈릴 때**  
  - `03-docker-cheatsheet.pdf`  
    → 자주 쓰는 명령어, 컨테이너 상태 확인, 에러 해결 힌트를 빠르게 찾을 수 있는 치트시트입니다.

- **OpenClaw의 성격·역할을 정의할 때**  
  - `04-system-prompt-template.pdf`  
    → “이 AI가 어떤 비서/에이전트인지”를 정의하는 문장을 작성할 때 기본 템플릿으로 활용합니다.

- **파일 자동 분류·업무 자동화 아이디어가 필요할 때**  
  - `05-file-classification-prompts.pdf`  
    → 다양한 분류/정리 작업에 바로 써볼 수 있는 프롬프트 예시를 모아둔 문서입니다.
