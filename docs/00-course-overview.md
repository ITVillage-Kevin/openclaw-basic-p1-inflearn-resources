# OpenClaw.ai 기초 1편 개요

이 문서는 **OpenClaw.ai 기초 1편** 강의 전체 흐름과, 이 Git 저장소를 어떻게 활용하면 좋은지 한눈에 보여주는 개요 가이드입니다.

---

## 1. 강의 목표

이 강의의 최종 목표는 아래 세 가지입니다.

1. 내 PC에 **안전한 AI 작업실(샌드박스)** 환경을 구축한다.
2. **Docker + Gemini API + OpenClaw.ai**를 연동하여 나만의 자율형 AI 에이전트를 띄운다.
3. 실제 파일을 대상으로 **분석·요약·보고서 생성**까지 자동화하는 첫 업무 흐름을 완성한다.

---

## 2. 강의 대상

이 강의는 다음과 같은 분들을 위해 설계되었습니다.

- IT 관련 초급자 또는 비전공자이지만, AI를 **업무 효율 도구로 활용**하고 싶은 분
- Docker나 터미널이 낯설지만, **단계별 가이드를 따라갈 의지**가 있는 분
- 단순 챗봇이 아니라, **“내 컴퓨터에서 직접 일을 해주는 AI 비서”**를 만들고 싶은 분

---

## 3. 섹션 구성 요약

강의는 총 5개 섹션으로 진행됩니다.

1. **섹션 1. 오리엔테이션**  
   - 에이전트 개념, OpenClaw.ai 소개, 하드웨어 전략, 설치 전 체크리스트 정리

2. **섹션 2. 환경 구축: 안전한 AI 작업실**  
   - Docker 설치, 샌드박스 개념, 프로젝트 폴더 구조, 볼륨 마운트 설정

3. **섹션 3. 두뇌 장착: Gemini API 연동**  
   - Google AI Studio에서 API 키 발급, OpenClaw 컨테이너 부팅, 초기 설정 점검

4. **섹션 4. 실전 실습: 자율적 파일 제어**  
   - OpenClaw 정체성 설정, 폴더 자동 탐색, 데이터 요약 및 보고서 자동 생성

5. **섹션 5. 마무리**  
   - 전체 흐름 복습, 유지보수 팁, 다음 단계(원격 제어·텔레그램 연동) 예고

---

## 4. 이 저장소의 역할

이 Git 저장소는 강의 속 실습을 **그대로 재현할 수 있도록** 돕는 보조 자료 모음입니다.  

주요 제공 내용은 다음과 같습니다.

- `docs/`  
  - 설치 체크리스트, Docker 가이드, 명령어 치트시트, 프롬프트 템플릿 등 **텍스트 기반 무료 가이드**

- `lesson-files/`  
  - 각 수업에서 사용하는 **연습용 문서, 테스트 데이터, 입력 파일**

- `releases/`  
  - 수업 단계별로 완성된 **`docker-compose.yml` 등 실행에 필요한 최소 파일 세트**

- `samples/`  
  - 강의에서 기대하는 **결과 예시(보고서 예시, 출력 예시 등)**

- `slides/`  
  - 강의에서 사용한 슬라이드 자료

- `templates/`  
  - `openclaw.json`, 시스템 프롬프트, 사용자 프로필 기본 양식 등 **재사용 가능한 템플릿**

---

## 5. 권장 학습 흐름

강의를 들을 때는 아래 순서를 추천합니다.

1. 인프런에서 해당 수업 영상을 시청한다.
2. 필요할 때 `docs/`의 가이드를 열어 설치·명령어·프롬프트를 참고한다.
3. 해당 수업 번호와 맞는 `lesson-files/` 폴더에서 실습 파일을 준비한다.
4. `releases/`에서 현재 수업 단계에 해당하는 실행 파일 세트를 사용한다.
5. 실습이 잘 되었는지 헷갈릴 때는 `samples/`의 예시 결과와 비교한다.
6. 개인 설정을 다듬고 싶을 때는 `templates/`를 참고해 자신만의 설정 파일을 만든다.

---

## 6. 선행 지식 및 준비물

- 복잡한 프로그래밍 지식은 필요하지 않습니다.
- 아래 정도의 기본 준비가 있으면 더욱 수월합니다.
  - 폴더 이동, 파일 복사/이동, 압축 풀기 등 **기본적인 PC 사용 능력**
  - 터미널(명령 프롬프트, 파워셸, 맥 터미널)에서 **명령어를 따라 입력해본 경험**이 조금이라도 있으면 좋습니다.

- 권장 준비물:
  - 인터넷이 되는 PC 또는 노트북
  - Docker Desktop 설치가 가능한 운영체제
  - Google 계정 (Gemini API 키 발급용)

## 7. 자주 묻는 질문(FAQ) 요약

- **Q. Docker나 터미널이 처음인데 따라갈 수 있을까요?**  
  A. 각 단계마다 예시·무료 가이드 문서가 제공되므로, 동영상과 문서를 함께 보며 차근차근 따라오면 됩니다.

- **Q. 중간에 실패해도 다시 처음부터 해야 하나요?**  
  A. `releases/` 폴더에 수업 단계별 실행 파일 세트가 있으므로, **해당 단계 상태로 바로 점프**해서 다시 시도할 수 있습니다.
