# Section 4 Lesson 14 – "이 폴더에는 뭐가 있지?" 파일 목록 자율 탐색

이 수업에서는 OpenClaw에게 **특정 폴더를 스스로 탐색하게 시키는 경험**을 해 봅니다.  
직접 파일을 하나씩 열어보는 대신, AI 비서에게 “이 폴더 안에 뭐가 있는지 먼저 살펴봐 줄래?”라고 부탁하는 단계입니다.

---

## 이 수업의 목표

- OpenClaw가 **지정된 폴더를 직접 스캔**하고, 내부에 어떤 파일들이 있는지 파악하게 한다.
- 폴더 탐색을 위해 어떤 도구/명령이 사용되는지, 로그와 결과를 통해 감각을 익힌다.
- 다음 수업(보고서 자동 생성)을 위한 준비 단계로, **폴더 구조와 파일 이름을 한 번에 머릿속에 그려보는 것**이 목표입니다.

---

## 사전 준비 사항

이 수업을 시작하기 전에 아래 상태가 준비되어 있어야 합니다.

1. 이전 수업까지 진행하면서 OpenClaw 컨테이너가 **정상 실행 중**이어야 합니다.
2. `workspace` 경로가 올바르게 설정되어 있고, 그 안에 **수업 14에서 사용할 샘플 폴더 및 파일 3개**가 준비되어 있어야 합니다.
   - 예:  
     ```bash
     workspace/
     └─ cafe_project/
        ├─ 신사업 기획안.txt
        ├─ 창업 예산안.csv
        └─ 프로젝트 회의록.txt
     ```
     (실제 파일명/구성은 강의에서 안내하는 예시를 따릅니다.)
3. OpenClaw 대시보드(웹 UI)에 접속 가능한 상태여야 합니다.

---

## 이 폴더에 포함된 내용

```bash
section04-lesson14/
└─ workspace/
   └─ cafe_project/
      ├─ (샘플 파일 1)
      ├─ (샘플 파일 2)
      └─ (샘플 파일 3)
```

- `workspace/cafe_project/`

  - 이 수업에서 **OpenClaw**가 탐색할 **샘플 프로젝트 폴더**입니다.

  - 실제 이름과 파일 구성은 강의에서 설명하는 예시를 따릅니다.

  - 다음 수업(섹션 4 수업 15)에서 이 폴더를 기반으로 **최종 보고서 자동 생성**까지 진행하게 됩니다.

⚠️ 이 수업에서는 .env, docker-compose.yml, openclaw.json 등을 새로 수정하지 않습니다.
여기 있는 것은 **“폴더 탐색 대상이 될 샘플 데이터”**입니다.

---

## 대시보드에서 할 일
1. OpenClaw 대시보드의 채팅 입력창을 엽니다.

2. 강의에서 안내하는 프롬프트(예:
"**workspace/cafe_project** 폴더 안에 어떤 파일들이 있는지 먼저 살펴보고, 파일 목록과 간단한 설명을 알려줘."
와 같은 형태)를 입력합니다.

3. OpenClaw가 스스로 폴더를 스캔하고,

- 어떤 파일이 있는지

- 각 파일이 어떤 역할을 할 것 같은지를 요약해서 알려주는지 확인합니다.

(정확한 프롬프트 문장은 강의 슬라이드 또는 별도 텍스트 파일에서 제공하는 내용을 따라 주세요.)

## 이 수업 이후의 상태
이 수업이 끝나면:

- OpenClaw가 `workspace` 내 특정 폴더를 스스로 탐색하고 이해할 수 있다는 감각을 얻게 됩니다.

- 수업 15에서 같은 `cafe_project` 폴더를 대상으로 **기획안 요약 + 예산 총합 계산 + 회의록 할 일 정리 + 최종_보고서.txt 생성**까지 한 번에 맡길 준비가 완료됩니다.

이 레슨은 “AI에게 폴더를 먼저 살펴보게 하는 습관”을 만드는 단계라고 생각하시면 됩니다.
