# Slides 안내

이 폴더에는 **OpenClaw.ai 기초 1편** 각 수업에서 사용한 슬라이드 자료(PDF)가 들어 있습니다.  
파일명은 `basic-p1-sectionXX-lessonYY.pdf` 형식을 따릅니다.

---

## 파일명 규칙

- `basic-p1-section01-lesson01.pdf`  
  - 기초 1편, 섹션 1, 수업 1 슬라이드
- `basic-p1-section01-lesson02.pdf`  
  - 기초 1편, 섹션 1, 수업 2 슬라이드
- `basic-p1-section02-lesson05.pdf`  
  - 기초 1편, 섹션 2, 수업 5 슬라이드  
- 그 외 파일들도 모두 같은 규칙을 따릅니다.  
  - `sectionXX` : 인프런 커리큘럼의 섹션 번호  
  - `lessonYY`  : 해당 섹션 내 수업 번호

---

## 어떻게 활용하면 좋을까요?

- **영상 수업을 보기 전**  
  - 슬라이드를 먼저 훑어보면서 이번 수업에서 다룰 키워드를 가볍게 파악합니다.
- **영상 수업을 보는 중**  
  - 터미널 명령어, 다이어그램, 폴더 구조처럼 화면을 일시정지하고 보고 싶은 내용이 있을 때,  
    해당 레슨의 PDF를 열어 확대해서 확인합니다.
- **영상 수업을 다 본 뒤**  
  - 복습할 때 핵심 슬라이드만 다시 보면서 전체 흐름을 정리하거나,  
    회사/개인 환경에 적용할 때 참고 자료로 활용할 수 있습니다.

---

## 찾아보기 팁

- 인프런에서 현재 듣고 있는 **섹션/수업 번호**를 확인한 뒤,  
  같은 번호의 `basic-p1-sectionXX-lessonYY.pdf` 파일을 찾으면 됩니다.
- 예: 인프런에서 “섹션 2, 수업 5”를 듣는 중이라면  
  `basic-p1-section02-lesson05.pdf` 파일을 열어보세요.
